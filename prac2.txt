# Installing Sublister

a. Clone the Github repository
1) cd home/kali/
2) cd /opt
3) sudo git clone https://github.com/aboul3la/Sublist3r.git

b. Install the requirements
1) cd Sublist3r/
2) pip install -r requirements.txt

c. If you want to manually install the dependencies
1) sudo pip install requests
2) sudo pip install dnspython
3) sudo pip install argparse

d. To run the tool
1) ./sublist3r.py

e. To create the symbolic link
1) sudo ln -sfv /opt/Sublist3r/sublist3r.py /usr/bin/sublist3r

f. To see the usage of sublist3r
1) sublist3r -h

g. To test sublist3r
1) sublist3r -v -d kali.org -t 5 -e bing -o /Desktop/subresult.txt

# Installing Maltego

a. To install Maltego
1) sudo apt install maltego

b. Run Maltego
1) Go to the searchbar of kali and search for maltego.
2) When you log into Maltego for the first time, you will have to accept some agreements and sign into your verified maltego account.
3) In the Configure Maltego window, once you reach point 6 -> Enter the Web browser as Firefox
4) Once you have successfully signed in, Select Footprint L1 machine followed by the name of the website that you want to mine.(cyberhia.com)
5) It will provide you with a tree like output.

# Google Hacking Database
a. To search for plaintext passwords or poorly configured Wordpress sites
1) inurl:/wp-content/uploads/ ext:txt "username" AND "password" | "pwd" | "pw"

b. To search for any vulnerable web servers
1) inurl:/proc/self/cwd

c. To search for any Open FTP servers
1) intitle:"index of" inurl:ftp

d. To search for any Email Lists
1) filetype:xls inurl:"emails.xls"

e. To search for any live cameras
1) inurl:top.htm inurl:currenttime -> IP based cameras
2) intitle:"webcamXP 5" -> Webcam-XP based transmission
3) inurl:"lvaapl.htm" -> General Live cameras

f. To search for any MP3, Movie, and PDF files
1) intitle: index of mp3 -> For various mp3 files.
2) intitle: index of mp4 OR intext: .mp4 -> For various mp4 files.
3) intitle: index of pdf -> For various PDF files.

g. To search for government documents
1) allintitle: restricted filetype:doc site.gov
